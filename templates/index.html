Input Format Guide:
  • Array/List: Comma-separated values → "5,3,1,2"
  • Binary Search: Array,target → "5,3,1,2,3" (searches for 3)
  • Single Value: For n (fibonacci, n-queens) → "8"
  • LCS (Longest Common Subsequence): Two strings separated by semicolon → "AGGTAB;GXTXAYB" (finds longest common subsequence)
  • Three Lists: values;weights;capacity → "60,100;10,20;50"
  • Kruskal/Prim (MST): First line = n (vertices), then edges (u v w) → "4\n0 1 1\n0 2 2\n1 2 3\n2 3 4"
  • Strassen Matrix Multiply: Two 2×2 matrices (4 rows) → "1 2\n3 4\n5 6\n7 8" (first 2 rows = Matrix A, next 2 rows = Matrix B)
  • Matrix Chain Multiply: Dimension array → "10 20 30 40" (for matrices 10×20, 20×30, 30×40)
  • TSP (Traveling Salesman Problem): Distance matrix rows separated by newline → "0 10 15 20\n10 0 35 25\n15 35 0 30\n20 25 30 0" (finds shortest tour visiting all cities)
